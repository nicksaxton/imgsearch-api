API - Image Search Abstraction Layer

About

A simple API that allows you to search for image data through the Google Custom Search API.

Search

You can search one of two ways, either a simple query like this:

https://sheltered-escarpment-81635.herokuapp.com/imagesearch/free code camp

or with a result offset like this:

https://sheltered-escarpment-81635.herokuapp.com/imagesearch/free code camp?offset=25

Recent Searches

You can also view the 10 most recent searches made with the API by entering the following URL:

https://sheltered-escarpment-81635.herokuapp.com/latest

Powered by Express and MongoDB
